# 金融市場資料分析作品集

> 申請東吳大學經濟學系用作品整理。主軸是用 Python、資料清理、回測、樣本外測試與風險驗證，展示我如何把生醫訓練中的證據思維轉到金融市場與經濟分析。

## 專案簡介

這份作品集整理三個自學與實作案例：

| 案例 | 主題 | 工具 | 可展示能力 |
|---|---|---|---|
| ETHUSDT 兩年回測 | Hurst / HMM 濾網與策略比較 | Python、CSV、SVG | 模型比較、參數測試、績效與回撤分析 |
| US100 一分鐘資料研究 | 五年資料下載、清理、壓力測試 | Python、Dukascopy、Monte Carlo | 資料工程、風險驗證、樣本外觀念 |
| XAUUSD 五年壓力測試 | 黃金策略穩定性與 walk-forward | Python、CSV、壓力測試 | 嚴格驗證、成本假設、結論判斷 |

## 核心結論

1. 單純看回測報酬不夠，必須同時檢查最大回撤、交易次數、成本壓力、Monte Carlo 與樣本外表現。
2. ETHUSDT 的 Hurst 濾網在本次測試中讓最佳組合由 BASE 22.05% 提升到 24.65%，但仍需要更多市場與期間驗證。
3. US100 的嚴格壓力測試顯示多數候選在成本與樣本外檢查下失敗，這反而證明研究流程有風險意識，不是只挑好看的績效。
4. XAUUSD 五年資料中有高報酬候選，但 walk-forward 與壓力測試提醒不能把全樣本結果直接視為可交易結論。
