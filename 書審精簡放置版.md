# 金融市場資料分析作品集

> 完整作品集連結：`待放上 GitHub 後貼上連結`

## 作品簡介

我用 Python 整理 ETHUSDT、US100 與 XAUUSD 等金融市場資料，完成資料下載、清理、回測、Hurst/HMM 濾網比較、Monte Carlo 與 walk-forward 驗證。這份作品不是為了呈現交易成果，而是想說明我如何用資料驗證假設，並把生醫訓練中的證據思維延伸到經濟與金融市場分析。

## 摘要表

| 作品 | 我做了什麼 | 代表結果 | 學到什麼 |
|---|---|---|---|
| ETHUSDT Hurst/HMM 濾網研究 | 比較 BASE、Hurst、HMM proxy 對回測表現的影響 | HURST_200_055 報酬 24.65%，最大回撤 -8.10% | 模型要看報酬，也要看回撤與交易次數 |
| US100 一分鐘資料與壓力測試 | 下載五年資料，檢查缺漏，再做 70/30 split、成本壓力與 Monte Carlo | 整理約 165 萬筆資料，8 個候選在嚴格壓力下皆未通過 | 失敗結果也能幫助排除不穩定模型 |
| XAUUSD 五年穩定性檢查 | 比較全樣本績效與 walk-forward 結果 | 全樣本候選亮眼，但樣本外表現不穩 | 不能只看漂亮的歷史績效，要避免過度最佳化 |

## 結論

這份作品讓我理解，金融市場分析不能只追求單一漂亮數字，而是要建立可以被檢查的研究流程：取得資料、清理資料、建立假設、回測模型、比較結果，最後用壓力測試與樣本外驗證修正判斷。這也讓我更確定想進入經濟學系，系統性學習統計、計量經濟、金融市場與產業分析，讓自己的資料分析能力建立在更完整的經濟學基礎上。
